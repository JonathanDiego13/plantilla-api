### Documentation for testing
* To test all the apps at once: **python manage.py test**
* To test a particular app: **python manage.py test <app>**
* To test a particular file: **python manage.py test <dotted_path_to_tests_file>** e.g. apps.users.tests
* To run 1 test case: **python manage.py test <dotted_path_to_tests_file.test_class_name>**
* To run just 1 test: **python manage.py test <dotted_path.test_class.test_function>**
